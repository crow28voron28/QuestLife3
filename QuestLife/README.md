# 🎮 QuestLife - Полная версия с API интеграцией

Android приложение для геймификации жизни с подключением к бесплатным базам данных питания и фитнеса!

## 🌟 Новые возможности

### 📡 **Интеграция с бесплатными API**

#### 🍽️ **Питание - 4 источника данных**

1. **Open Food Facts API** (полностью бесплатный)
   - База: 2+ миллиона продуктов со всего мира
   - URL: https://world.openfoodfacts.org/
   - Особенности: поиск по названию, сканирование штрих-кодов
   - Данные: БЖУ, калории, размер порции

2. **USDA FoodData Central API** (бесплатный)
   - База: официальная база питания США
   - URL: https://fdc.nal.usda.gov/
   - Регистрация: бесплатный API ключ
   - Особенности: очень подробные данные о нутриентах

3. **Edamam Nutrition API** (бесплатный tier)
   - Анализ рецептов по ингредиентам
   - URL: https://developer.edamam.com/
   - Особенности: рассчитывает БЖУ для блюд

4. **Локальная база** (35+ продуктов)
   - Работает оффлайн
   - Популярные продукты

#### 💪 **Фитнес - 3 источника данных**

1. **Wger Workout Manager API** (полностью бесплатный)
   - База: 300+ упражнений
   - URL: https://wger.de/
   - Особенности: упражнения + готовые программы тренировок
   - Не требует API ключа!

2. **ExerciseDB API** (бесплатный tier на RapidAPI)
   - База: 1300+ упражнений с GIF демонстрацией
   - URL: https://rapidapi.com/justin-WFnsXH_t6/api/exercisedb
   - Особенности: анимированные GIF, подробные описания
   - Бесплатный tier: 100 запросов/месяц

3. **Локальная база** (30+ упражнений)
   - Работает оффлайн
   - Базовые упражнения с инструкциями

### 📊 **Расширенная база квестов**

#### ✅ **Ежедневные квесты** (500+)
- 🏃 Физические: 150 квестов (бег, тренировки, спорт)
- 🧠 Умственные: 100 квестов (чтение, обучение, работа)
- ❤️ Здоровье: 100 квестов (питание, сон, вода)
- 👥 Социальные: 80 квестов (общение, помощь)
- 🎨 Творческие: 70 квестов (хобби, искусство)

**Награды**: стандартные (10-200 XP, 5-100 золота)

#### 📅 **Еженедельные квесты** (100+)
- Более сложные цели на неделю
- Награда: **XP × 3, Золото × 3** 💰
- Примеры:
  - "Тренироваться 5 раз в неделю" → 150 XP, 75 золота
  - "Пробежать 20 км за неделю" → 300 XP, 150 золота
  - "Без сахара всю неделю" → 150 XP, 75 золота

#### 📆 **Ежемесячные квесты** (50+)
- Глобальные цели на месяц
- Награда: **XP × 10, Золото × 10** 💎
- Примеры:
  - "Тренироваться 20 дней в месяце" → 2000 XP, 1000 золота!
  - "Прочитать 4 книги" → 1000 XP, 500 золота
  - "Похудеть на 2 кг" → 1000 XP, 500 золота

```kotlin
// Ежемесячные (3 из 50+) с десятикратной наградой

```

### Repository паттерн

```kotlin
class NutritionRepository {
    suspend fun searchFoods(query: String): List<Food>
    suspend fun findByBarcode(barcode: String): Food?
    suspend fun analyzeRecipe(ingredients: List<String>): NutritionData?
}

class ExerciseRepository {
    suspend fun getAllExercises(): List<Exercise>
    suspend fun getExercisesByBodyPart(bodyPart: String): List<Exercise>
    suspend fun getWorkoutPlans(): List<WorkoutProgram>
}

class QuestRepository {
    fun generateDailyQuests(count: Int): Flow<List<HealthQuest>>
    fun generateWeeklyQuests(count: Int): Flow<List<HealthQuest>>
    fun generateMonthlyQuests(count: Int): Flow<List<HealthQuest>>
}
```

## 🚀 Установка и настройка

### 1. Получите бесплатные API ключи

#### Open Food Facts - НЕ ТРЕБУЕТ КЛЮЧ! ✅
Работает сразу из коробки!

#### USDA FoodData Central (опционально)
1. Перейдите на https://fdc.nal.usda.gov/api-key-signup.html
2. Зарегистрируйтесь (бесплатно)
3. Получите API ключ
4. Вставьте в `USDAFoodDataAPI`:
```kotlin
private const val API_KEY = "ВАШ_КЛЮЧ_ЗДЕСЬ"
```

#### ExerciseDB на RapidAPI (опционально)
1. Зарегистрируйтесь на https://rapidapi.com/
2. Подпишитесь на ExerciseDB (бесплатный tier)
3. Скопируйте X-RapidAPI-Key
4. Вставьте в `ExerciseDBAPI`:
```kotlin
private const val API_KEY = "ВАШ_RAPIDAPI_КЛЮЧ"
```

#### Edamam Nutrition (опционально)
1. Зарегистрируйтесь на https://developer.edamam.com/
2. Создайте приложение
3. Получите APP_ID и APP_KEY
4. Вставьте в `EdamamAPI`:
```kotlin
private const val APP_ID = "ВАШ_APP_ID"
private const val APP_KEY = "ВАШ_APP_KEY"
```

### 2. Обновите build.gradle

Используйте `build_with_api.gradle` вместо стандартного `build.gradle`

### 3. Обновите AndroidManifest.xml

Используйте `AndroidManifest_with_permissions.xml` для разрешений сети и камеры

### 4. Добавьте сканер штрих-кодов (опционально)

Для сканирования продуктов по штрих-коду добавьте ML Kit Barcode Scanning.
